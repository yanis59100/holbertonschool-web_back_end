# Holberton School Web Back-End Projects

This repository contains the back-end web development projects I completed during my studies at Holberton School.  
The focus of these projects was to build server-side applications, work with databases, and implement RESTful APIs.

## Projects Overview
- Building RESTful APIs with Node.js and Express  
- Database integration using MySQL and MongoDB  
- Server-side scripting and application logic  
- Authentication and user management  
- Connecting front-end with back-end services  

## Tech / Stack
- Node.js / Express  
- MySQL / MongoDB  
- JavaScript (Node.js)  
- Linux environment  
- Git for version control  

## What I Learned
- Designing and building server-side applications  
- Creating and managing databases  
- Implementing RESTful API endpoints  
- Integrating front-end and back-end components  
- Writing clean, maintainable, and scalable back-end code  

## Usage
Clone the repository and explore individual project folders to see source code, instructions, and examples of back-end applications.
